===============
django-starsweb
===============

A hosting app for the classic 4X space strategy game Stars!
(http://en.wikipedia.org/wiki/Stars%21).


Requirements
------------
Python >= 2.6
Django >= 1.2
django-template-utils (https://bitbucket.org/ubernostrum/django-template-utils)
lxml
starslib (not yet released)

Recommended
-----------
django-micro-press
